# Работа с ветками  

### Клонирование чужого репозитория  
git clone <ссылка на репозиторий>  

### Создание веток  
git branch <название ветки>  

### Навигация по веткам  
git branch - показать все ветки репозитория  
git checkout <название ветки> - переключиться на ветку  

### Сравнение веток  
git diff <название ветки 1> <название ветки 2>  
git diff <>~N <> - показать разницу между коммитом, который был N коммитов назад и текущим  

### Удаление веток  
git branch -d <название ветки> - удалить ветку, если она является частью main  
git branch -D <название ветки> - удалить ветку, даже если она не объединена с main  

### Слиянение веток  
git merge <название ветки> - объединить ветку <название ветки> с текущей активной веткой  

### Работа с удалённым репозиторием  
git push -u origin <название ветки> - отправить новую ветку <название ветки> в удалённый репозиторий и связать локальную ветку с удалённой  
git pull - подтянуть изменения текущей ветки из удалённого репозитория
